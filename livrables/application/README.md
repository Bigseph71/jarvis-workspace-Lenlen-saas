# Application

## Ce que contient ce dossier

- Specs fonctionnelles et cahiers des charges
- Code source et prototypes
- Architecture technique et schémas
- Documentation technique et guides d'utilisation
- Décisions techniques et notes de conception

## Lien avec tes objectifs

Couvre tes deux projets SaaS actifs : le **développement d'un logiciel SaaS** et la **mise en place d'une plateforme d'agents**.

## Attention aux secrets

Le code de ce dossier peut nécessiter des clés d'API (VAPI, Retell AI, hébergeurs cloud, etc.). Ne jamais coder les clés en dur dans les fichiers. Utilise des variables d'environnement, documentées dans le [.env.example](../../.env.example) à la racine.
